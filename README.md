# 💊 PharmaX - Online Medical Store

PharmaX is a web-based online medical store that allows users to browse medicines, upload prescriptions, and place medicine orders conveniently. The project is developed using HTML, CSS, JavaScript, and PHP.

---

## 📌 Features

- 🏠 Home Page
- 🔐 User Login
- 💊 Browse Medical Products
- 🛒 Add Products to Cart
- 📄 Upload Prescription
- ❤️ Healthcare Product Categories
- 📱 Simple and Responsive User Interface

---

- PHP

## 🚀 How to Run the Project

### Using VS Code

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension.
4. Right-click **home.html**.
5. Click **Open with Live Server**.

### Using XAMPP (for PHP)

1. Install XAMPP.
2. Copy the project folder to:

```
C:\xampp\htdocs\
```

3. Start **Apache** from the XAMPP Control Panel.
4. Open your browser and visit:

```
http://localhost/medicin/home.html
```
